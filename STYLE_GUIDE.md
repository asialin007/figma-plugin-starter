# 🎨 批量重命名插件 - 界面美化说明

## ✨ 美化内容

### 1. 渐变背景
- **紫色渐变背景**：从 `#667eea` 到 `#764ba2`
- **12px 圆角**：现代化的卡片设计

### 2. 标题样式
- **20px 大标题**，600 字重
- **✨ Emoji 装饰**：增加视觉吸引力
- **负字间距**：更紧凑的排版

### 3. 输入框美化
- **2px 边框**，默认灰色 `#e2e8f0`
- **聚焦效果**：
  - 边框变为主题紫色
  - 添加 3px 外阴影（半透明）
- **8px 圆角**
- **过渡动画**：0.2s 平滑过渡

### 4. 复选框优化
- **浅灰色背景** `#f7fafc`
- **悬停效果**：背景变深
- **内边距**：12px 舒适间距
- **自定义文字样式**

### 5. 按钮设计

#### 主按钮（应用重命名）
- **紫色渐变背景**
- **🚀 Emoji 图标**
- **阴影效果**：`0 4px 6px` 半透明阴影
- **悬停动画**：
  - 向上移动 2px
  - 阴影加深
- **点击反馈**：回到原位

#### 次要按钮（取消）
- **白色背景**
- **灰色边框** `#e2e8f0`
- **悬停效果**：背景变浅灰

### 6. 分隔线
- **渐变效果**：透明 → 灰色 → 透明
- **优雅的视觉分隔**

### 7. 间距层次
- **表单组间距**：16px
- **按钮组间距**：8px
- **整体内边距**：24px

## 🛠️ 自定义样式

### 修改主题颜色
编辑 `src/styles.css`，找到以下变量：

```css
/* 主渐变色 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 改成蓝色主题 */
background: linear-gradient(135deg, #4299e1 0%, #667eea 100%);
```

### 修改圆角大小
```css
/* 输入框和按钮圆角 */
border-radius: 8px;  /* 改为 12px 更圆润 */

/* 容器圆角 */
border-radius: 12px; /* 改为 16px 更圆润 */
```

### 修改按钮阴影
```css
/* 主按钮阴影 */
box-shadow: 0 4px 6px rgba(102, 126, 234, 0.2);

/* 加强阴影 */
box-shadow: 0 8px 16px rgba(102, 126, 234, 0.3);
```

## 📱 响应式设计建议

如果需要支持不同窗口大小：

```css
/* 小窗口 */
.plugin-container {
  padding: 16px; /* 减少内边距 */
}

/* 大窗口 */
@media (min-height: 600px) {
  .plugin-container {
    padding: 32px; /* 增加内边距 */
  }
}
```
